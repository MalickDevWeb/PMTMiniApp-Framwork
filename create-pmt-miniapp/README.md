# create-pmtminiapp

CLI de création de projets Mini Program basés sur le starter officiel `PMTMiniApp`.

Nom npm :
- `create-pmtminiapp`

## Usage

```bash
npx create-pmtminiapp mon-app
```

ou

```bash
npm create pmtminiapp mon-app
```

## Options

```bash
npx create-pmtminiapp mon-app --no-install
```

## Maintenance du template

Le template est régénéré depuis :
- `../from-scratch-app`

Commande :

```bash
npm run refresh:template
```
