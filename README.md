# PMTMiniApp-Framwork

![PMTMiniApp Dev Tools](assets/pmtminiapp-dev-tools.png)

Ce depot contient deux briques principales :

- `from-scratch-app` : framework mini-app de base
- `create-pmt-miniapp` : CLI pour generer un projet a partir du framework

## Structure

- [create-pmt-miniapp](create-pmt-miniapp)
- [from-scratch-app](from-scratch-app)

## CI

Le workflow GitHub Actions valide :

- le framework `from-scratch-app`
- le package CLI `create-pmt-miniapp`
- un projet genere en smoke test
