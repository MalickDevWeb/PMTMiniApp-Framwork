# Étape 04 — Routing et paramètres URL

## Objectif
Passer des données contrôlées entre pages via navigation.

## Travaux
1. `navigateTo` avec query params
2. lecture via `onLoad(options)`
3. règles d'encodage `encodeURIComponent/decodeURIComponent`

## Critères
- valeur transmise fiable
- aucune dépendance à un état implicite
