# Composants du projet

Ces composants viennent de la base `tcmpp-boilerplate`, mais ils ont été repris avec un nommage plus explicite et une API plus simple.

Composants disponibles :
- `app-bouton-action`
- `app-carte-section`
- `app-champ-texte`
- `app-barre-navigation`
- `app-boite-dialogue`
- `app-chargeur-page`
- `app-message-action`
- `app-etat-vide`

Règle simple :
- le nom du composant doit dire ce qu'il affiche ou ce qu'il fait
- la page ne doit pas dépendre de détails cachés
- les propriétés doivent rester compréhensibles sans ouvrir le fichier
